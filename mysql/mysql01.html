MySQL

MySQL은 오픈 소스의 관계형 데이터베이스 관리 시스템(RDBMS)입니다.

MySQL은 다중 사용자 및 다중 스레드를 지원하는 관계형 데이터베이스 관리 시스템(RDBMS)으로, 데이터의 안정성과 보안성을 제공합니다. 대부분의 운영체제에서 작동하며, 웹 애플리케이션 개발, 데이터 분석, 빅 데이터, 클라우드 기반 애플리케이션 등 다양한 분야에서 널리 사용되고 있습니다. 사용이 쉽고 확장성이 높아서 많은 개발자들이 선호하는 데이터베이스 시스템 중 하나로 MySQL은 SQL(Structured Query Language)을 사용하여 데이터를 관리하는데 이는 데이터베이스에 저장된 데이터를 조작하기 위한 표준적인 언어로서, 데이터를 검색, 조작, 정의하는 데 사용되는 선언형 언어입니다. SQL문을 통해 사용자는 어떤 데이터를 원하는지를 기술하고, 데이터베이스 시스템은 이를 처리하는 방법을 결정합니다.

MySQL 설치

MAMP는 macOS에서 웹 사이트를 개발할 때 사용되는 솔루션 스택으로, macOS (M), Apache (A), MySQL (M), PHP (P)의 약어입니다. 이는 웹 개발을 위한 환경을 편리하게 설정하고 운영할 수 있는 도구로, 개발자들이 로컬 환경에서 웹 애플리케이션을 개발, 테스트 및 디버깅할 때 유용하게 사용됩니다. MAMP를 사용하면 개발자들은 자신의 컴퓨터에서 Apache 웹 서버, MySQL 데이터베이스 서버, 그리고 PHP를 실행할 수 있습니다. 이를 통해 웹 개발 프로세스를 효율적으로 관리하고 개발 환경을 손쉽게 구축할 수 있습니다.

윈도우 : cd MAMP/bin/mysql/bin (cd c:\MAMP\bin\mysql\bin)
