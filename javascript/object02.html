01. find()

find() 메서드는 배열 내에서 주어진 판별 함수를 만족하는 첫 번째 요소의 값을 반환합니다.

  • 배열에서 첫 번째로 조건을 만족하는 요소를 반환합니다.
  • 사용자가 제공한 콜백 함수에 따라 검색 조건을 설정할 수 있습니다.
  • 조건을 만족하는 요소가 없는 경우 undefined를 반환합니다.
  • 원본 배열을 변경하지 않습니다.
                   
    const numbers = [10, 20, 30, 40, 50];

    const result = numbers.find(function(number) {
        return number > 25;
    });
    
    console.log(result); // 30
결과 확인하기
30

02. forEach()

forEach() 메서드는 배열의 각 요소에 대해 주어진 함수를 실행합니다. 이 메서드는 배열을 반복하면서 각 요소를 처리하고, 요소마다 주어진 함수를 실행합니다.
또한 요소뿐만 아니라 해당 요소의 인덱스와 배열 자체도 반복한다는 것입니다. 이 메서드의 콜백 함수는 세 가지 인자를 받을 수 있는데, 첫 번째는 현재 처리 중인 요소의 값(currentValue)이고, 두 번째는 해당 요소의 인덱스(index)이며, 세 번째는 배열 자체(array)입니다. 이러한 특징을 이용하면 배열의 각 요소를 반복하면서 해당 요소의 인덱스나 배열 전체에 접근할 수 있고, 각 요소와 해당 요소의 인덱스를 함께 출력할 수도 있습니다.

  • 사용자가 제공한 콜백 함수에 따라 각 요소를 처리하고 원하는 작업을 수행할 수 있습니다.
  • 원본 배열을 변경하지 않고, 새로운 배열을 반환하지 않습니다.
  • 배열의 첫 번째 요소부터 마지막 요소까지 순차적으로 처리됩니다.
                
    const fruits = ['apple', 'banana', 'cherry'];

    fruits.forEach(function(item) {
        console.log(item);
    });
결과 확인하기
apple
banana
cherry

03. filter()

filter() 메서드는 배열의 각 요소를 조건에 따라 걸러내어 새로운 배열을 생성합니다. 이 함수는 일종의 배열 메서드이며, 콜백 함수를 사용하여 각 요소를 평가하고, 조건을 만족하는 요소만으로 이루어진 새로운 배열을 반환합니다.

  • 배열의 각 요소에 대해 주어진 함수의 테스트를 수행하고, 조건을 만족하는 요소만을 새로운 배열에 담으며, 원본 배열은 변경되지 않습니다.
  • 사용자가 제공한 콜백 함수에 따라 각 요소를 평가하고, 조건을 만족하는 요소만을 새 배열에 추가합니다.
  • 조건에 맞는 요소만 필터링: 콜백 함수가 true를 반환하는 요소들만 새로운 배열에 포함됩니다.
  • 순서 보존: 새로운 배열은 원본 배열의 순서를 유지합니다.
                
const numbers = [1, 2, 3, 4, 5];

const evenNumbers = numbers.filter(function(number) {
return number % 2 === 0;  //배열 numbers에서 짝수만을 필터링하여 evenNumbers라는 새로운 배열을 생성
});

console.log(evenNumbers); // [2, 4]
결과 확인하기
[2, 4]

04. map()

map() 메서드는 배열의 각 요소에 대해 지정된 함수를 호출하여 새로운 배열을 생성합니다. 이 새로운 배열은 원래 배열의 각 요소에 함수를 적용한 결과로 이루어집니다.

  • 각 요소에 대해 실행할 함수로, 콜백 함수는 세 가지 매개변수를 받습니다.
  • currentValue: 배열에서 현재 처리 중인 요소입니다.
  • index (선택사항): 현재 처리 중인 요소의 인덱스입니다.
  • array (선택사항): map() 메서드가 호출된 배열입니다.
  • thisArg (선택사항): 콜백 함수 내에서 this로 사용할 값입니다.
                
    const numbers = [1, 2, 3, 4, 5];

    const squaredNumbers = numbers.map(number => number * number);

    console.log(squaredNumbers); // [1, 4, 9, 16, 25]
결과 확인하기
[1, 4, 9, 16, 25]

05. includes()

includes() 메서드는 배열이 특정 요소를 포함하는지 여부를 판별하는 메서드입니다. 이 메서드는 주어진 값이 배열에 존재하면 true를 반환하고, 그렇지 않으면 false를 반환합니다.

  • array.includes(searchElement[, fromIndex]) 기본구문
  • searchElement: 배열에서 찾을 요소입니다.
  • fromIndex (선택사항): 검색을 시작할 인덱스입니다. 기본값은 0입니다. 만약 음수 값을 넣으면 배열 끝에서부터 검색을 시작합니다.
  • includes() 메서드는 주어진 searchElement가 배열에 포함되어 있는지를 확인합니다. 만약 포함되어 있다면 true를 즉시 반환하고, 그렇지 않으면 false를 반환합니다.
  • 일치 여부를 판단할 때 일치하는 요소의 값과 타입을 모두 고려합니다. 따라서 엄격한 일치 비교(===)를 사용합니다.
  • 배열의 원시값 뿐만 아니라 객체도 비교할 수 있습니다. 객체의 경우, 참조 값이 같은 경우에만 true를 반환합니다.
                
    const array = [1, 2, 3, 4, 5];

    console.log(array.includes(3)); // 출력: true
    console.log(array.includes(6)); // 출력: false
결과 확인하기
true
false

06. indexOf()

indexOf() 메서드는 배열에서 지정된 요소를 찾아 그 인덱스를 반환하는 메서드입니다. 만약 배열에 해당 요소가 존재하지 않으면 -1을 반환합니다.

  • 배열에서 요소를 찾을 때 사용됩니다.
  • 첫 번째 일치하는 요소의 인덱스를 반환합니다.
  • 검색을 시작할 위치를 지정할 수 있습니다. 따라서 배열의 특정 부분에서만 검색할 수 있습니다.
  • 배열의 원시값 뿐만 아니라 객체도 비교할 수 있습니다. 하지만 엄격한 일치 비교(===)를 사용하므로 타입도 일치해야 합니다.
                
    const array = [1, 2, 3, 4, 5];

    const index1 = array.indexOf(3);
    console.log(index1); // 출력: 2

    const index2 = array.indexOf(6);
    console.log(index2); // 출력: -1
결과 확인하기
2
-1

07. join()

join() 메서드는 배열의 각 요소를 지정된 구분자로 구분하여 하나의 문자열로 합칩니다. 구분자를 지정하지 않으면 기본값인 쉼표로 요소들이 구분됩니다. 

                
    const array = ['apple', 'banana', 'orange'];

    const result = array.join();
    console.log(result); // 출력: "apple,banana,orange"
    

    const array = ['apple', 'banana', 'orange'];

    const result = array.join(' ');
    console.log(result); // 출력: "apple banana orange"
결과 확인하기
"apple,banana,orange"
"apple banana orange"

08. push()

push() 메서드는 배열의 끝에 하나 이상의 요소를 추가하는 메서드입니다. 이 메서드는 배열의 길이를 변경하고, 새로운 요소를 추가한 후 변경된 배열의 새로운 길이를 반환합니다.

  • 배열의 끝에 지정된 요소를 추가합니다. 추가되는 요소의 순서는 배열에 추가되는 순서대로 유지됩니다.
  • 주로 스택(Stack) 자료구조를 구현하는 데 사용됩니다.
                
    const array = [1, 2, 3];

    array.push(4);
    console.log(array); // 출력: [1, 2, 3, 4]

    array.push(5, 6);
    console.log(array); // 출력: [1, 2, 3, 4, 5, 6]
결과 확인하기
[1, 2, 3, 4]
[1, 2, 3, 4, 5, 6]

09. reduce()

reduce() 메서드는 배열의 각 요소에 대해 지정된 콜백 함수를 실행하여 하나의 결과값을 얻는 메서드입니다. 이 메서드는 배열을 순회하면서 각 요소를 처리하고, 그 결과를 누적하여 하나의 값으로 반환합니다.

  • 배열의 각 요소에 대해 콜백 함수를 순차적으로 실행하며 결과값을 누적합니다.
  • 누적된 결과값은 다음 콜백 호출의 첫 번째 인수로 전달됩니다.
  • 초기값을 제공하지 않으면 배열의 첫 번째 요소가 초기값이 됩니다.
  • 콜백 함수에서 반환하는 값이 누적됩니다.
  • 주로 배열을 가공하거나 요소를 결합하여 단일 결과값을 얻는 데 사용됩니다.
                
    const array = [1, 2, 3, 4, 5];

    const sum = array.reduce((accumulator, currentValue) => accumulator + currentValue, 0);

    console.log(sum); // 출력: 15
결과 확인하기
15

10. reverse()

reverse() 메서드는 배열의 순서를 반전시키는 메서드입니다. 즉, 배열의 요소들을 역순으로 재배열합니다.

  • 새로운 배열을 만들지 않고 기존 배열을 수정합니다.
                
    const array = [1, 2, 3, 4, 5];

    array.reverse();
    console.log(array); // 출력: [5, 4, 3, 2, 1]
결과 확인하기
[5, 4, 3, 2, 1]

11. slice()

slice() 메서드는 배열의 일부분을 추출하여 새로운 배열로 반환하는 JavaScript의 내장 메서드입니다. 이 메서드는 기존 배열을 변경하지 않고, 지정된 시작 인덱스부터 끝 인덱스 바로 전까지의 요소들을 복사하여 새로운 배열을 생성합니다.

  • array.slice([start[, end]]) 기본구문
  • start (선택사항): 추출을 시작할 인덱스입니다. 생략하면 인덱스 0부터 시작합니다. 음수 값을 지정하면 배열의 끝에서부터 인덱스를 계산합니다.
  • end (선택사항): 추출을 끝낼 인덱스 바로 전의 인덱스입니다. 생략하면 배열의 끝까지 추출합니다. 음수 값을 지정하면 배열의 끝에서부터 인덱스를 계산합니다. 이 인덱스에 해당하는 요소는 포함되지 않습니다.
                
    const array = ['apple', 'banana', 'orange', 'grape', 'kiwi'];

    const slicedArray = array.slice(1, 4);
    console.log(slicedArray); // 출력: ["banana", "orange", "grape"]
결과 확인하기
["banana", "orange", "grape"]

12. splice()

splice() 메서드는 배열의 내용을 변경하고, 해당 위치에서 요소를 삭제하거나 새로운 요소를 추가하는 메서드입니다. 이 메서드는 배열을 수정하며, 수정된 배열을 반환합니다.

  • array.splice(start[, deleteCount[, item1[, item2[, ...]]]]) 기본구문
  • start: 변경을 시작할 인덱스입니다.
  • deleteCount (선택사항): 제거할 요소의 수입니다. 생략하면 start부터 배열 끝까지의 모든 요소가 제거됩니다.
  • item1, item2, ... (선택사항): 추가할 요소입니다. start 인덱스 위치부터 삽입됩니다.
  • splice() 메서드는 배열의 내용을 변경합니다. 지정된 start 인덱스부터 시작하여 deleteCount 만큼의 요소를 삭제하고, 이후의 요소들을 새로운 요소들로 대체합니다. 새로운 요소를 추가하려면 item1, item2, ...을 지정하면 됩니다.
                
    const array = [1, 2, 3, 4, 5];

    // 인덱스 1부터 2개의 요소를 삭제하고, 6과 7을 추가
    array.splice(1, 2, 6, 7);

    console.log(array); // 출력: [1, 6, 7, 4, 5]
결과 확인하기
[1, 6, 7, 4, 5]

13. sort()

sort() 메서드는 배열의 요소를 정렬하는 메서드입니다. 이 메서드는 배열을 정렬하여 변경하고, 변경된 배열을 반환합니다. 기본적으로 문자열로 변환된 각 요소를 유니코드 코드 포인트 순서로 정렬합니다.

  • array.sort([compareFunction]) 기본구문
  • compareFunction (선택사항): 정렬 순서를 정의하는 함수입니다. 생략하면 배열의 요소는 문자열로 변환된 후 유니코드 코드 포인트 순서로 정렬됩니다.
  • sort() 메서드는 배열의 요소를 정렬하여 변경합니다. 이때, 배열은 숫자 또는 문자열과 같은 기본 데이터 유형이거나 객체와 같은 복합 데이터 유형을 포함할 수 있습니다. compareFunction을 제공하면 배열의 요소를 비교하는 방법을 지정할 수 있습니다.
  • compareFunction은 두 개의 인수를 받는 함수로, 정렬 순서를 결정합니다. 반환 값이 0보다 작으면 a를 b보다 앞에 놓고, 0보다 크면 b를 a보다 앞에 놓고, 0이면 순서를 변경하지 않습니다.
                
    const array = [3, 1, 4, 1, 5, 9, 2, 6, 5];

    array.sort((a, b) => a - b);

    console.log(array); // 출력: [1, 1, 2, 3, 4, 5, 5, 6, 9]
결과 확인하기
[1, 1, 2, 3, 4, 5, 5, 6, 9]
