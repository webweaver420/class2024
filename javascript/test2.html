01. 변수 : 데이터 불러오기

변수는 값을 저장하고 참조하기 위한 기본적인 요소로, 저장되어 있는 값은 나중에 필요할 때 다시 사용할 수 있습니다. 

{
    let x = 100;
    let y = 200;
    let z = "javascript";

    console.log(x);  // lat 변수 x값에 저장되어 있는 값을 출력
    console.log(y); 
    console.log(z); 
}
결과 확인하기
100
200
javascript

02. 상수 : 데이터 불러오기

상수는 일반적으로 데이터를 저장하고 관리하는 방법 중 하나로 프로그램 실행 중에 변경되지 않는 값을 나타내며, 일반적으로 데이터를 하드코딩하여 사용할 때 유용합니다. 

{
    const x = 100, y = 200, z = "javascript"; 

    console.log(x, y, z);  
}
결과 확인하기
100
200
javascript

03. 배열 : 데이터 불러오기

배열은 대괄호([]) 안에 쉼표로 구분된 값들의 목록으로 정의되고, 여러 유용한 메서드들이 내장되어 있어 요소를 추가, 제거, 정렬, 검색 등 다양한 작업을 수행할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];  // 'arr' 배열 선언, 배열의 요소는 인덱스를 사용하여 접근, 0부터 시작

    console.log(arr[0], arr[1], arr[2]); 
}
결과 확인하기
100
200
javascript

04. 배열 : 데이터 불러오기 : 갯수

배열의 length 속성을 활용하여 해당 배열에 저장된 데이터의 개수를 파악할 수 있습니다. 

{
    const arr = [100, 200, "javascript"];  // 'arr' 배열 정의, 배열 안에는 숫자 100, 200, javascript 를 포함하고 있음

    console.log(arr.length);  // 'length' 속성은 배열 내 요소의 개수를 나타냄
}

05. 배열 : 데이터 불러오기 : for()문

for문을 사용하여 배열의 각 요소에 접근하고, 배열의 모든 요소를 반복하여 출력할 수 있습니다. 

{                
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let i = 0; i < arr.length; i++) {               
    console.log(arr[i]);
    }
}

초기값 let i = 0;에서 시작
i가 배열의 길이(arr.length)보다 작은 동안 반복
반복이 진행될 때마다 i는 1씩 증가함

06. 배열 : 데이터 불러오기 : forEach()

배열의 forEach() 메서드는 배열의 각 요소에 대해 주어진 함수를 실행하는 메서드입니다.
이 메서드를 사용하면 반복문을 사용하지 않고도 배열의 각 요소를 순회하며 작업을 수행할 수 있습니다. 

{      
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    //01
    arr.forEach(function (element) {                      
    console.log(element);
    })
    //02
    arr.forEach(function (element, index) {
    console.log(index);
    })
    //03
    arr.forEach(function (element, index, array) {
    console.log(array);
    })          
}

*첫 번째 코드 블록에서는 각 요소의 값(element)을 출력
*두 번째 코드 블록에서는 각 요소의 인덱스(index)를 출력
*세 번째 코드 블록에서는 배열 자체(array)를 출력

07. 배열 : 데이터 불러오기 : for of

for of 를 통해 배열의 각 요소를 반복하며, 반복할 때마다 현재 요소를 지정된 변수에 할당합니다. for of는 요소의 값을 직접 가져와 반복하기 때문에 코드가 간결하고 가독성이 높아집니다. 

{          
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let element of arr) {   // arr 배열의 각 요소를 반복하여 처리하는 for...of 반복문임
        console.log(element);   //  배열 arr의 각 요소를 반복하며, 각 요소를 element 변수에 할당하여 그 값을 콘솔에 출력함
    }    
}

08. 배열 : 데이터 불러오기 : for in

for in 반복문을 사용해서 배열의 인덱스를 반복하고, 각 인덱스에 해당하는 요소를 참조할 수 있습니다. 

{          
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let element in arr) {
    console.log(arr[element]);  // 배열 arr의 인덱스를 순회하면서, 변수 element에 인덱스를 할당하고, arr[element]를 통해 해당 인덱스의 배열 요소 값을 콘솔에 출력
    }
}

for in을 배열에 사용할 경우, 배열의 인덱스가 문자열로 처리되며, 배열 이외의 열거 가능한 속성도 순회할 수 있기 때문에 예상치 못한 결과를 초래할 수 있습니다.
따라서 배열을 순회할 때는 주로 for of를 사용하는 것이 좋습니다.

09. 배열 : 데이터 불러오기 : map()

map() 메서드는 배열의 각 요소에 대해 주어진 함수를 호출한 결과를 모아서 새 배열로 반환합니다.
호출된 배열을 변형시키지 않으며, 원본 배열과 동일한 길이의 새 배열을 생성합니다. 

{          
    const arr = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    //01
    arr.map(function (element) {
        console.log(element)
    });    

    arr.map(function (element, index) {
        console.log(index)
    });
    
    arr.map(function (element, index, array) {
        console.log(array)
    });
}
