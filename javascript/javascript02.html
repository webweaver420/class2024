01. 변수 : 데이터 불러오기

변수는 값을 저장하는 데 사용되며, 이를 통해 프로그램이 데이터를 처리하고 조작할 수 있습니다.
변수는 다양한 유형과 이름을 가질 수 있으며, 값을 변경할 수도 있습니다.
예를 들어, "x"라는 변수에 숫자 5를 할당할 수 있습니다. 그러면 이 변수는 이제 5라는 값을 가지게 됩니다. 이후에 프로그램에서는 "x"를 참조하여 5를 사용하거나 변경할 수 있습니다.
변수는 프로그램의 유연성을 제공하며, 동일한 코드를 사용하여 다양한 데이터를 처리할 수 있도록 도와줍니다. 

{                   
    num x = 100;
    num y = 200;
    num z = "javascript";

    console.log(x); 
    console.log(y); 
    console.log(z); 
}
결과 확인하기
100
200
javascript

02. 상수 : 데이터 불러오기

상수(Constant)는 변수와 비슷한 개념이지만 변수는 값을 여러 번 할당하고 변경할 수 있는 반면에 상수는 값이 한번 할당되면 그 값을 변경할 수 없습니다.
즉, 상수는 변하지 않는 값을 나타내는 데 사용되며, 프로그램 전체에서 고정된 값을 사용해야 할 때 유용합니다. 

{
    const x = 100, y = 200, z = "javascript"; 

    console.log(x, y, z);  
}
결과 확인하기
100
200
javascript

03. 배열 : 데이터 불러오기 : 여러값

배열은 대괄호 [] 안에 쉼표로 구분된 데이터 요소들을 순서대로 저장하는 자료 구조입니다. 배열은 javascript에서 가장 일반적으로 사용되는 데이터 구조 중 하나이며, 많은 유용한 메서드들이 내장되어 있어 요소를 추가, 제거, 검색, 정렬 등 다양한 작업을 수행할 수 있습니다. 각 요소는 배열 내의 위치를 나타내는 인덱스를 사용하여 개별적으로 접근할 수 있습니다.
인덱스는 0부터 시작하여 배열의 길이보다 1 작은 값까지의 범위를 가집니다. 이를 통해 특정 요소에 접근하거나 해당 요소의 값을 변경할 수 있습니다. 

{
    const num = [100, 200, "javascript"];  

    console.log(num[0], num[1], num[2]); 
}

100
200

05. 배열 : 데이터 불러오기 : for()문

06. 배열 : 데이터 불러오기 : forEach()

배열의 forEach()는 대부분의 프로그래밍 언어에서 사용되는 반복문의 한 형태입니다. 주로 배열 또는 리스트와 같은 컬렉션의 각 요소에 대해 반복 작업을 수행할 때 사용됩니다.

    });

    //03 배열 전체 값 출력
    num.forEach(function (element, index, array) {

07. 배열 : 데이터 불러오기 : for of

for of 를 통해 배열의 각 요소를 반복하며, 반복할 때마다 현재 요소를 지정된 변수에 할당합니다.

08. 배열 : 데이터 불러오기 : for in

for...in 반복문을 사용하여 배열의 각 요소의 인덱스를 반복하고, 해당 인덱스를 사용하여 요소를 출력합니다. 그러나 for...in 반복문은 주로 객체의 속성을 순회할 때 사용되며, 배열의 요소를 순회하기에는 적합하지 않습니다. 배열의 요소를 순회할 때는 for...of 반복문을 사용하는 것이 더 적절합니다. for...in 반복문은 배열의 인덱스뿐만 아니라 상속된 속성까지 열거할 수 있으며, 순서가 보장되지 않을 수 있습니다. 따라서 배열을 순회할 때 for...in을 사용하는 것은 권장되지 않습니다. 

{          
    const num = [100, 200, 300, 400, 500, 600, 700, 800, 900];

    for (let element in num) {
    console.log(num[element]); 
    }
}

09. 배열 : 데이터 불러오기 : map()

배열의 map() 메서드는 배열의 각 요소에 대해 주어진 함수를 호출한 결과를 모아서 새 배열로 반환합니다. 호출된 배열을 변형시키지 않으며, 원본 배열과 동일한 길이의 새 배열을 생성합니다. "메서드"는 프로그래밍에서 사용되는 용어로, 특정한 작업을 수행하기 위한 코드 블록이나 함수를 가리킵니다.

10. 배열 : 데이터 불러오기 : filter()

filter() 메서드는 배열 내의 각 요소에 대해 주어진 함수를 호출하여 조건을 만족하는 요소만으로 이루어진 새로운 배열을 반환합니다. 이 메서드는 주로 배열을 필터링하고 특정 조건을 충족하는 요소들을 추출할 때 사용됩니다. 

{
    const num = [1, 2, 3, 4, 5, 6, 7, 8, 9,];

    num.filter(function (element) {
        console.log(element);
    });

    num.filter(function (element, index) {
        console.log(index);
    });

    num.filter(function (element, index, array) {
        console.log(array);
    });

    const arr = num.filter(function (element) {
        return element > 5;
    });
    console.log(arr);

    const arr1 = num.filter = ((element) => {
        return element > 5;
    });
    console.log(arr);

    const arr2 = num.filter = (element => {
        return element > 5;
    });
    console.log(arr);

    const arr3 = num.filter = (element => element > 5);

    console.log(arr);        
}

11. 배열 : 데이터 불러오기 : 배열 구조 분해 할당

배열 구조 분해 할당(Array Destructuring Assignment)은 JavaScript에서 사용되는 기능 중 하나로, 배열을 분해하여 그 값을 개별 변수에 할당하는 것을 말합니다. 이를 통해 배열의 각 요소에 접근하여 변수에 저장할 수 있습니다. 

{                    
    // 기본 변수값 할당
    const arr = [100, 200, "javascript"];
    const [a, b, c] = arr;

    console.log(a);
    console.log(b);
    console.log(c);
    }
    // 선언에서 분리한 할당
    {
    const [a, b, c] = [100, 200, "javascript"];

    console.log(a); 
    console.log(b);
    console.log(c);
    }
    {
    // 기본 값 설정
    const [a, b, c = "javascript"] = [100, 200];

    console.log(a);
    console.log(b);
    console.log(c);
    }
    {
    // 배열 일부 요소 무시
    const [a, , c] = [100, 200, "javascript"];

    console.log(a);
    console.log(c);
    }
    {
    // 나머지 값 할당
    const [a, b, ...rest] = [100, 200, "javascript", "jquery"];

    console.log(a); 
    console.log(b); 
    console.log(...rest); 
    }
    {
    // 변수 값 교환
    let a = 100, b = 200;
    [a, b] = [b, a];

    console.log(a);
    console.log(b);
    }
    {
    //다차원 배열 펼침연산자
    const [a, [b, c]] = [1, [2, "javascript"]];

    console.log(a);
    console.log(b);
    console.log(c);                    
}

12. 배열 : 데이터 불러오기 : 배열 펼침 연산자(Spread Operator)

배열 펼침 연산자(Spread Operator)는 JavaScript의 문법 중 하나로, 배열을 펼쳐서 개별 요소로 분리하는 역할을 합니다. 이 연산자는 주로 배열을 복사하거나 다른 배열과 결합할 때 , 함수 호출시 사용됩니다. 배열 펼침 연산자는 코드를 더 간결하게 만들고, 배열을 다룰 때 유용한 기능 중 하나입니다 

{                   
    // 배열 복사
    const arr = [100, 200, "javascript"];
    const copy = [...arr];

    console.log(arr); 
    console.log(...arr);
    console.log(copy);
    }
    {
    // 배열 합치기
    const arr1 = [100, 200];
    const arr2 = [300, 400];
    const merged1 = [...arr1, ...arr2];
    const merged2 = [...arr1, "javascript", ...arr2, "jquery"];

    console.log(arr1.concat(arr2));
    console.log(merged1);
    console.log(merged2);
    }
    {
    // 함수 인자로 사용하기
    function numbers(a, b, c) {
        return a + b + c;
    }
    const nums = [100, 200, 300];
    console.log(numbers(...nums));
    }
    {
    // 문자열 배열로 변환하기
    const text1 = "javascript";
    const text2 = [...text1];

    console.log(text1);
    console.log(text2);                      
}

13. 객체 : 데이터 불러오기 : 기본

객체를 데이터로 불러오기 위해서는 해당 객체의 속성(키-값 쌍)에 접근하는 방법을 사용합니다. JavaScript에서 객체의 속성에 접근하는 방법은 점 표기법과 중괄호 표기법 두 가지가 있습니다.
점 표기법: 객체의 속성에 접근할 때는 객체 이름 뒤에 점(.)을 붙이고 그 뒤에 속성의 이름을 적습니다.
중괄호 표기법: 객체의 속성에 접근할 때는 객체 이름 뒤에 대괄호{}를 사용하고 그 안에 속성의 이름을 문자열 형태로 적습니다. 

{               
    const obj = {
    a: 100,
    b: 200,
    c: "javascript"
    }

    console.log(obj.a);
    console.log(obj.b);
    console.log(obj.c);

    console.log(obj["a"]);
    console.log(obj["b"]);
    console.log(obj["c"]);                           
}
결과 확인하기
100
200
javascript

14. 객체 : 데이터 불러오기 : Object.keys()

Object.keys()는 JavaScript의 내장 함수 중 하나로, 주어진 객체의 속성 이름들을 배열로 반환합니다. 이 함수는 객체의 속성 이름들을 반복하여 배열로 반환하므로, 객체의 속성을 순회하거나 객체의 속성들에 대한 작업을 수행할 때 유용하게 사용됩니다. 

{               
    const obj = {
    a: 100,
    b: 200,
    c: "javascript"
    }

    console.log(Object.keys(obj));  // 키값을 출력             
}
결과 확인하기
["a", "b", "c"]

Object.values()는 JavaScript의 내장 함수 중 하나로, 주어진 객체의 속성 값들을 배열로 반환합니다. 이 함수는 객체의 속성 값을 반복하여 배열로 반환하므로, 객체의 속성 값들에 대한 작업을 수행할 때 유용하게 사용됩니다. 

{               
    const obj = {
    a: 100,
    b: 200,
    c: "javascript"
    }
    console.log(Object.values(obj));       
}

16. 객체 : 데이터 불러오기 : Object.entries()

Object.entries()는 JavaScript의 내장 함수 중 하나로, 주어진 객체의 속성들을 [키, 값] 쌍의 배열로 반환합니다. 이 함수는 객체의 각 속성에 대해 순회하며 배열의 형태로 [키, 값] 쌍을 생성하여 배열에 저장합니다. 

{               
    const obj = {

17. 객체 : 데이터 불러오기 : Object.assign()

Object.assign()은 JavaScript의 내장 함수 중 하나로, 하나 이상의 소스 객체에서 대상 객체로 속성을 복사할 때 사용됩니다. 이 함수는 대상 객체를 반환하며, 소스 객체의 속성들이 대상 객체에 복사됩니다. 만약 대상 객체에 이미 같은 이름의 속성이 있다면, 해당 속성은 덮어씌워집니다. 

{               
    const obj1 = {
    a: 100,
    b: 200,
    }
    const obj2 = {
    c: 300,
    d: 400,
    }
    const obj3 = Object.assign(obj1, obj2);

    console.log(obj3);
}
결과 확인하기
{a: 100, b: 200, c: 300, d: 400}

18. 객체 : 데이터 불러오기 : hasOwnProperty()

hasOwnProperty()는 자바스크립트 객체의 메소드 중 하나입니다. 이 메소드는 객체가 특정 속성(키)을 직접 소유하고 있는지 여부를 나타내는 부울 값을 반환합니다.
보통 자바스크립트 객체는 상속된 속성도 포함하여 객체의 모든 속성을 확인할 때 사용합니다. 그러나 때로는 상속된 속성을 제외하고 객체가 직접 소유한 속성만을 확인해야 하는 경우가 있습니다. hasOwnProperty() 메소드를 사용하면 상속된 속성은 확인하지 않고 직접 소유한 속성만을 검사할 수 있습니다. 

{               
    const obj = {
        a: 100,
        b: 200,
        c: "javascript"
    }
    console.log(obj.hasOwnProperty("a"));     
    console.log(obj.hasOwnProperty("b"));     
    console.log(obj.hasOwnProperty("c"));     
    console.log(obj.hasOwnProperty("d"));     

    // in  연산자
    console.log("a" in obj)   
    console.log("b" in obj)   
    console.log("c" in obj)  
    console.log("d" in obj)   
}

false
true

true

21. 객체 : 데이터 불러오기 : 객체 펼침연산자

객체 펼침연산자는 객체의 속성을 새로운 객체에 펼쳐서 복사하는 역할을 합니다. 이를 통해 쉽게 새로운 객체를 만들거나 기존 객체를 병합할 수 있습니다.
객체 리터럴 내에서 다른 객체를 펼침, 함수의 매개변수로 객체를 전달할 때 펼침, 배열의 요소로 객체를 전달할 때 펼침 세가지 상황에서 사용될 수 있습니다. 

{               
    const obj = { a: 100, b: 200, c: "javascript" };
    const copy = { ...obj };

    console.log(obj);
    console.log(copy);
    }
    {
    const obj1 = { a: 100, b: 200, }
    const obj2 = { c: 300, d: 400 }
    const merge1 = { ...obj1, ...obj2 }
    const merge2 = { ...obj1, x: "javascript", ...obj2, y: "jquery" }

    console.log(obj1);
    console.log(merge1);
    console.log(merge2); 
}
결과 확인하기

{ a: 100, b: 200}
{ a: 100, b: 200, c: 300, d: 400}
{ a: 100, b: 200, x: "javascript" c: 300, d: 400, y:"jquery"}

22. 서버 데이터 불러오기 : XMLHttpRequest

XMLHttpRequest는 JavaScript에서 네트워크 요청을 보내는 객체입니다. 이를 통해 서버로부터 데이터를 비동기적으로 요청하고, 응답을 처리할 수 있습니다.
XMLHttpRequest는 오래된 방법이므로 현대적인 웹 개발에서는 fetch() API나 더 높은 수준의 라이브러리 또는 프레임워크를 사용하는 것이 일반적입니다. 

{               
var xhr = new XMLHttpRequest();
xhr.open("GET", "https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json", true);

xhr.onreadystatechange = function () {
    if (xhr.readyState === 4 && xhr.status === 200) {
        var responseData = xhr.responseText;
        var jsonData = JSON.parse(responseData);

        console.log(jsonData);
    } else if (xhr.readyState === 4) {
        console.error("데이터 불러오기 오류: " + xhr.status);
    }
};

xhr.send();
}
결과 확인하기
60개의 값

23. 서버 데이터 불러오기 : fetch API

fetch API는 JavaScript에서 네트워크 요청을 보내는 방법 중 하나로, XMLHttpRequest보다 간단하고 강력한 API입니다. fetch API를 사용하면 Promise를 기반으로하여 비동기적으로 서버 데이터를 요청하고 응답을 처리할 수 있습니다. 

{               
    fetch("https://webstoryboy.github.io/webs2024/json/gineungsaJC2011_05.json")
    .then(response => {
        return response.json();
    })
    .then(data => {
        console.log(data);
    })
    .catch(error => {
        console.error("데이터 불러오기 오류:", error);
    })
}
결과 확인하기
60개의 값
